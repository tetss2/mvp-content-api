# Task Blocks

Prioritized backlog for high-impact visible improvements. Each block should fit in roughly 2-4 hours and avoid architecture expansion.

## P0: Forbidden Patterns

Add explicit forbidden phrases and structures to the active generation prompt. Add a simple cleanup pass for repeated generic AI phrases.

Visible result: posts sound less like generated advice and more like a real Telegram author.

## P0: Emotional Cadence

Improve generation instructions so posts move from felt experience to explanation to one practical shift. Avoid abrupt advice.

Visible result: warmer, more psychologically believable content.

## P0: Few-Shot Examples

Inject a few short Dinara-like examples for openings, transitions, endings, and CTAs.

Visible result: less templated structure and more author identity.

## P0: Humanization Pass

Add light post-processing after generation:

- remove obvious AI transitions
- reduce forced markdown issues
- normalize paragraph spacing
- prevent numbered-list shape
- soften repetitive endings

Visible result: cleaner text ready for Telegram.

## P0: Anti-AI Cleanup

Reduce or eliminate:

- "важно понимать"
- "следует отметить"
- "таким образом"
- "в современном мире"
- "данная проблема"
- essay-like conclusions
- generic therapy disclaimers where not needed

Visible result: less robotic voice.

## P1: Telegram Regeneration Controls

Add one-tap regeneration variants:

- softer
- more practical
- more Dinara-like

Visible result: fewer manual edit loops before publishing.

## P1: Paragraph Rhythm Improvements

Tune prompt and cleanup rules so paragraphs are short, varied, and readable on mobile.

Visible result: posts feel native to Telegram.

## P1: Dinara Opening Styles

Create a compact set of opening patterns:

- quiet observation
- client-room insight
- gentle contradiction
- direct emotional naming
- soft question

Visible result: less repetitive starts.

## P1: CTA Realism

Replace generic calls to action with softer endings:

- "Замечаете у себя такое?"
- "С чем это у вас откликается?"
- "Можно начать с одного честного вопроса к себе..."

Visible result: less salesy, more human interaction.

## P1: Emotional Transitions

Improve transitions between empathy, explanation, and practice.

Visible result: posts stop feeling stitched together.

## P2: Sexologist Style Cleanup

Make sexologist modes less theatrical and less emoji-heavy while keeping warmth, safety, and clarity.

Visible result: more credible expert voice on intimate topics.

## P2: Publish Preview Hygiene

Improve Telegram preview text and captions so photo/video posts keep the strongest part of the text without awkward cuts.

Visible result: better final publishing quality.

## P2: Feedback-to-Regeneration Shortcut

When a user taps "not voice" or "weak expertise", offer a direct regeneration option using that critique.

Visible result: faster correction loop.
