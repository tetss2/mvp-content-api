# Claude Export

## Purpose

This folder contains exported knowledge, summaries and architectural decisions from Claude projects.

---

## What will be stored here

- chat summaries
- prompts
- workflows
- architecture notes
- deployment decisions
- bot logic
- AI behavior settings
- reusable instructions

---

## Export Strategy

Each important Claude conversation should be converted into a structured markdown summary.

Recommended structure:

- problem
- decisions
- implementation
- current status
- next steps

---

## Goal

Build a centralized AI-native knowledge base independent from a single chat platform.
